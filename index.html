This app needs a Firebase project to sync between everyone. It takes ~2 minutes:

  1. Open the Firebase console and create a project.
  2. Build → Firestore Database → Create database → Start in test mode.
  3. Project settings → Your apps → Web (</>) and register an app.
  4. Copy the firebaseConfig values into firebase-config.js.

Reload once you've pasted your config.
